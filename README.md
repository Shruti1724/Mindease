# MindEase 🧠🌿

MindEase is a mental wellness web application designed to help users manage stress, practice mindfulness, and maintain emotional well-being.

## Features
- User authentication (login & register)
- Guided breathing and meditation pages
- Journaling feature with database storage
- Immersive calming audio environments

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file (see `.env.example`)
4. Run the server: `node server.js`
